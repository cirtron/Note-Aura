# Note-Aura

**セルフホスト型 AI ナレッジ受信箱 — 何でも放り込んで、いつでも問い返せる。**

URL を貼り付け、メールを転送し、画像をアップロードするか、直接メモを入力するだけ。
バックグラウンドの AI が自動でタイトル・要約・タグ・埋め込みを生成します。
自然言語で**ノートに質問**（RAG ＋引用元表示）するか、キーワード検索でいつでも取り出せます。

単一の Go バイナリ＋SQLite で動作。デフォルトはローカルの [Ollama](https://ollama.com) モデルを使用。
データはサーバーの外へ出ません。OpenAI 互換エンドポイントにも対応しています。

## 機能

- **取り込み → 処理 → 取り出しのループ。** 保存は即時完了。バックグラウンドのワーカープールが重い AI 処理を担い、完了後にノートを「処理中」から「完了」へ切り替えます。AI が利用できない場合もコンテンツは保存されます（ノートに**再試行**ボタン表示）。
- **取り込み時に自動整理** — AI がタイトル・要約・タグ・**カテゴリ**を自動生成（空欄のフィールドのみ）。要約言語は手動選択またはコンテンツに合わせて自動判定。
- **Web リンク・YouTube・Facebook の取り込み** — リンクを貼り付けるだけでページ全文または動画の文字起こしを取得して要約。JavaScript レンダリングページにはヘッドレスブラウザでのフォールバックも対応。Facebook 投稿・動画・リールをサポート。管理者は `cookies.txt` を設定して認証が必要なコンテンツにも対応可能。
- **停止ボタン** — AI 処理中のノートをいつでもキャンセル可能。ノートは「停止済み」状態になり（エラーの「失敗」とは別）、後から再試行できます。
- **画像 OCR** — 写真やスクリーンショットをアップロードすると、ビジョンモデルがテキストを抽出します。
- **ファイルアップロード** — ロールごとに許可するファイル形式を設定。テキストファイルはノート本文に、その他はダウンロード可能な添付ファイルとして保存。
- **メール → ノート** — ユーザーごとに専用のプラスアドレスが割り当てられ、そこへ送ったメールが自動的にノートになります（リンクだけのメールはリンク先ページを取得）。
- **ノートへの質問（RAG）** — 埋め込みベクトルによるセマンティック検索を経由して対話モデルが回答、引用元のノートも表示します。
- **マルチユーザー** — メール／パスワード認証、メール認証、外部サービス不要の内蔵 CAPTCHA（歪み文字式）、招待制。ノートの個別共有（読み取り専用または編集可能）とグループ（副管理者・読み書き権限設定付き）に対応。
- **カレンダー** — ノートにイベント日時・開始／終了時刻・終日フラグを設定可能。月ビューと日別アジェンダを表示。メールリマインダーと各国の祝日にも対応。
- **Markdown エディタ**（EasyMDE）— リアルタイムプレビューとインライン画像に対応。goldmark でレンダリングし、bluemonday でサニタイズ。
- **整理と閲覧** — 階層型**カテゴリ**（`親/子`形式）、タグ、キーワード検索、多様な**並び順**オプション、ページサイズ可変のページネーション。スマートフォンではカテゴリ／タグフィルターをメニュー下の折りたたみ**フィルター**パネルに収納。
- **ノート管理** — 複数選択して**一括削除**、全ノートを JSON ファイルへエクスポート／インポート（設定から操作）。
- **ロールとクォータ** — ロールごとにストレージ・AI アクセス・1 日の上限・グループ／招待上限・許可アップロード形式を設定。ユーザー個別の上書きも可能。
- **プラグイン可能な AI バックエンド** — デフォルトはローカル **Ollama**。ユーザーごとに任意の OpenAI 互換エンドポイント（OpenAI・Gemini 互換・OpenRouter など）へ上書き可能。独自プロンプトのカスタマイズにも対応。
- **管理ダッシュボード** — ユーザーごとの使用状況・最新ノート・サーバーモニター。ユーザー管理（停止／削除）、ロール、ブランディング、モデルとプロンプト、祝日、登録設定、メール、**HTTPS のオン／オフ切り替え**を管理。
- **多言語 UI** — English／繁體中文／简体中文／日本語（デフォルトはブラウザ言語に従い、ヘッダーで切り替えてアカウントに記憶）。確認メール・パスワードリセット・招待メールは各受信者の言語で送信。

## 技術スタック

Go + Fiber、SQLite（`modernc.org/sqlite`、純 Go・CGO なし）、FTS5 全文検索、Go 実装のコサイン類似度ベクトル検索、`html/template` サーバーサイドレンダリング。

## 実行

1. [Ollama](https://ollama.com) をインストールしてデフォルトモデルを取得（後から設定で OpenAI 互換バックエンドに変更可能）：
   ```
   ollama pull llama3.1
   ollama pull nomic-embed-text
   ollama pull deepseek-ocr
   ```
2. 設定ファイルをコピーして編集：
   ```
   cp config.example.yaml config.yaml
   ```
3. ビルドして実行：
   ```
   go build -o note-aura.exe .
   ./note-aura.exe
   ```
4. http://localhost:8090 を開いてアカウントを登録（最初のアカウントが自動的に管理者になります）。

**メンテナンスツール**（プロジェクトフォルダで実行）：

```powershell
.\update.ps1     # Windows：新しいコードに更新、全データを保持（停止 → 再ビルド → 再起動）
.\reset.exe      # 全データを初期状態に戻す（ビルド：go build -o reset.exe ./cmd/reset/）
```

```bash
./update.sh      # Linux/macOS：同上（systemd を自動検出）
```

`update.ps1` / `update.sh` はデータベースやアップロードファイルを変更しません。`reset.exe` は完全に削除します（実行前にサーバーを停止してください）。詳細は [INSTALL.md](INSTALL.md#9-backup-upgrade--reset) を参照。

**HTTPS** で提供するには、`config.yaml` で証明書と秘密鍵（PEM 形式）を指定し、`session.secure: true` と `https://` の `base_url` を設定します：

```yaml
tls:
  cert_file: "certs/note-aura.crt"
  key_file:  "certs/note-aura.key"
```

> **オプション連携：** YouTube 取り込みには [`yt-dlp`](https://github.com/yt-dlp/yt-dlp) が `PATH` に必要。ヘッドレスブラウザのフォールバックには Chrome/Chromium が必要。メール→ノート・リマインダー・認証・招待には SMTP/IMAP の設定が必要。詳細は [INSTALL.md](INSTALL.md) を参照。

詳細なセットアップは **[INSTALL.md](INSTALL.md)**、使い方は **[USER_GUIDE.ja.md](USER_GUIDE.ja.md)** を参照してください。

## プロジェクト構成

```
main.go                 配線：設定・DB・ワーカープール・サーバー・メールポーリング
internal/
  config/   db/         YAML 設定；SQLite スキーマ＋クエリ（FTS 同期）
  auth/                 bcrypt＋セッショントークン
  ai/                   Provider インターフェース；Ollama＋OpenAI 互換実装
  ingest/               HTML→テキスト、URL 取得（JSON-LD＋ヘッドレス）、YouTube 文字起こし
  rag/                  チャンキング、埋め込み（逆）シリアライズ、コサイン類似度、top-k
  worker/               非同期ジョブパイプライン（取得/OCR → タイトル/要約/タグ/埋め込み）
  mailer/   reminder/   送信 SMTP；カレンダーリマインダースケジューラー
  emailin/              受信 IMAP → ノートポーリング
  server/               Fiber ルーティング＋ハンドラー
cmd/reset/              スタンドアロン「全データ削除」ツール
web/templates/          サーバーサイドレンダリングページ（Markdown エディタ）
update.ps1              全データを保持しながらバイナリをその場で更新
```
